Tags básicas

As tags em HTML são necessárias na criação de websites com textos, imagens e indexação de links internos ou externos.

A tag html identifica que o código está escrito em HTML.

A tag head contém os dados para execução do código.

Dentro da tag head é inserido o título do documento com a tag title.

Na tag body é inserida o conteúdo do site.

Atributos das tags

Os atributos nas tags customizam o conteúdo dos sites.

O atributo href combinada com a tag a aponta para links externos ou internos.

O atributo src com a tag img mostra de onde a imagem veio.

Também com a tag img os atributos width e height otimizam a largura e a altura de imagens.

Caso uma imagem não possa ser carregada é usado o atributo alt, mostra determinado texto.

Um pouco de tipografia

Os cabeçalhos em um documento HTML são formatados com os seguintes níveis: h1, h2, h3, h4, h5 e h6.

Para escrever parágrafos usa-se a tag p.

Uma tag útil para trabalhos acadêmicos e afins, é a blackquote utilizada para citar outros autores.

A tag mark é utilizada para grifar partes de um texto.

Tag i é representada como escrita inclinada para direita.

A tag u sublinha o texto.

A tag strong destaca partes do texto com negrito.

A tag hr separa com uma linha segmentos do código.

As tags sub e sup são definidas como subscrito e sobrescrito.

A tag font pode especificar a fonte de um texto como cor e tamanho.

Listas ordenadas e não ordenadas

Listas ordenadas imiciam e terminam com a tag ol, os itens são inseridos um a um com letras ou números após a tag li.

Já as listas não ordenadas os itens são geralmentes marcados (pontos), usa-se a tag ul.
